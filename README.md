# MiP ESP-01 pack

Stop! This is an untested commit.  It probably doesn't work. Once I test all of this, I'll update this documentation.

An [ESP-01](https://academy.cba.mit.edu/classes/networking_communications/ESP8266/esp01.pdf) adapter board for [WowWee Labs'](https://github.com/WowWeeLabs/) [MiP](https://wowwee.com/mip) robot.  Similar to the [MiP D1 mini pack](https://github.com/Tiogaplanet/MiP_D1-mini_pack), this board adapts the ESP-01 to MiP.  Install the ESP-01 to a pin header or solder it directly to the board and you'll be programming MiP to connect to the Internet in no time.
